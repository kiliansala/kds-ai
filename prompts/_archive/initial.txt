CONTEXTO GENERAL
----------------
Este repositorio NO es un proyecto nuevo.
Es la continuación de un proyecto creado inicialmente con Antigravity.

El objetivo del proyecto es construir un Design System AI-friendly donde:
- Figma es el Single Source of Truth (SSOT) visual
- Los componentes se generan y validan contra Figma
- Se parte de Web Components como base

Stack principal:

- Lit
- Storybook
- Dev app / documentation site
- Frameworks derivados: Angular, React y Blazor

- Código, ficheros, documentación técnica: inglés
