# Button

Buttons communicate actions that users can take. They are commonly used in forms, dialogs, and toolbars.

## Overview

KDS Buttons provide a set of accessible, easy-to-use button components for your application.
They come in several appearances to handle various emphasis levels: **filled**, **outlined**, **tonal**, **text**, and **elevated**.

## Variations

### Outlined Button

Use outlined buttons for medium-emphasis actions. They are important but not the primary action of the app.

<Canvas of={ButtonStories.Outlined} />

### Tonal Button

Tonal buttons are used for actions that need more emphasis than text buttons but less than filled buttons.

<Canvas of={ButtonStories.Tonal} />

### Button with Icon

Buttons can include an icon to provide visual context.
